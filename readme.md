# VyomaCSS

**VyomaCSS** is a lightweight, SCSS-based CSS framework built by **Prasoon Jadon** under **Vyoma Labs**.
It is designed for **indie developers** who want clean, readable, and extensible styles without unnecessary bloat.

VyomaCSS is available on **npm** and is completely **open-source**.
You can explore the source code and contribute via the GitHub repository.
